Pockets Infinity

Version: 0.1.0-alpha

Pockets Infinity gives players of the game Destiny enhanced access to the items on their Guardians and the Vault. Its basically Batman's utility belt for your Guardian, all your items in one place.

Since version 0.1.0-alpha is pre-release, Pockets Infinity only dipslays a limited amount of information.

Requirements

  • Chrome Desktop Web Browser

Getting Started

Download a .zip or clone the repo: git clone https://github.com/blainefricks/pockets-infinity.git.

Install dependencies: bower install or see the Wiki for a list of dependencies.

Enable Chrome Extensions Developer Mode and point it to the app/ folder.

Once you have installed the Chrome Extension, sign into Bunge.net and then press the Pockets Infinity icon or options page to launch.

Reporting Bugs

Have a bug to report? First search through the open and closed issues before submitting your bug. If your problem has not been submitted, please open a new issue and include the version of Pockets Infinity you are using.
